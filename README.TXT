PROJECT TITLE: Grid Plotter Using BlueJ
PURPOSE OF PROJECT: To make a rough simulation of a grid and filling the squares in it with different colored blocks
VERSION or DATE: 2014
HOW TO START THIS PROJECT: After opening in BlueJ, right-click "GridPlotterUsingBlueJ", then left-click
"void main(String[] Args)", then click "Ok"
AUTHORS: Noah Getz
USER INSTRUCTIONS: After making a "New Grid" with the defined amount of rows and columns, you can pick a pattern to fill the grid with
